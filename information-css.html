Вступ

CSS (Cascading Style Sheets) - це мова стилізації, яка використовується для опису вигляду та форматування HTML документів. CSS дозволяє розділити зміст веб-сторінки (HTML) від її представлення (CSS).

Структура CSS

Синтаксис


    селектор {
    властивість: значення;
    }

Перелік CSS Властивостей

  • background-color: Колір фону.
  • color: Колір тексту.
  • font-family: Шрифт.
  • font-size: Розмір шрифту.
  • font-weight: Товщина шрифту.
  • line-height: Висота рядка.
  • text-align: Вирівнювання тексту.
  • text-decoration: Декорація тексту.
  • margin: Зовнішній відступ.
  • padding: Внутрішній відступ.
  • border: Рамка.
  • border-radius: Радіус рамки.
  • display: Відображення елементу.
  • width: Ширина.
  • height: Висота.
  • position: Позиціонування.
  • top, left: Відступи для позиціонованих елементів.
  • float: Плаваюче вирівнювання.
  • opacity: Прозорість.
  • box-shadow: Тінь для блоку.
  • text-shadow: Тінь для тексту.
  • transform: Трансформація елементу.
  • transition: Плавний перехід між станами.
  • display: flex: Включення flex-контейнера.
  • justify-content: Вирівнювання flex-елементів.
  • text-overflow: Контролює, як текст переповнює свою властивість контейнера.
  • white-space: Визначає, як браузер поводиться щодо пробілів всередині елемента.
  • text-transform: Визначає перетворення тексту (наприклад, у верхній чи нижній регістр).
  • cursor: Визначає форму курсора при наведенні на елемент.
  • overflow: Контролює, як вміст, який перевищує розміри блока, відображається.
  • list-style: Загальний стиль для списку, який об'єднує всі стилі списку в одному властивості.
  • text-align-last: Вирівнювання останнього рядка або єдиного слова, якщо вони коротше за відповідний контекст.
  • word-break: Визначає, чи може браузер розривати слова, щоб їх вміщати в блок на деяших розмірах.
  • text-justify: Контролює інтервал між словами в рядках, коли зміст блока змінюється за рахунок вирівнювання.
  • text-indent: Визначає відступ першого рядка тексту в елементі.
  • box-sizing: Визначає, як браузер враховує ширину та висоту елемента, включаючи його поля та границі.
  • outline: Встановлює контур навколо елемента, який не займає простору у макеті.
  • pointer-events: Визначає, як елемент реагує на події курсора.
  • resize: Визначає, чи можна змінювати розмір елемента за допомогою користувача.
  • overflow-x: Контролює, як вміст, який перевищує ширину блока, відображається у горизонтальному напрямку.
  • overflow-y: Контролює, як вміст, який перевищує висоту блока, відображається у вертикальному напрямку.
  • unicode-bidi: Визначає напрямок тексту, якщо текстовий напрямок об'єкта відрізняється від текстового напрямку батьківського об'єкта.
  • overflow-wrap: Контролює перенесення слова в межах блока, коли вона не поміщається.
  • column-count: Визначає кількість колонок, які елемент розбивається на.
  • column-gap: Визначає відстань між колонками у множинних колонах елемента.
  • column-rule: Встановлює стиль, ширину та колір лінії, яка розділяє колонки елемента.
  • break-inside: Визначає, чи може бути елемент розділений на дві частини після обривання колонок, сторінок або областей в просторі.
  • column-span: Визначає, чи простягається елемент через всі колонки у множинних колонках або областях.
  • column-width: Визначає ширину колонок у множинних колонках елемента.
  • column-fill: Визначає, як розподіляється вміст між колонками у множинних колонках елемента.
  • columns: Об'єднує стилі для визначення кількості колонок, їх ширини та стилю лінії.
  • page-break-after: Визначає, чи слід відображати обрив сторінки після елемента.
  • page-break-before: Визначає, чи слід відображати обрив сторінки перед елементом.
  • page-break-inside: Визначає, чи може бути елемент розділений на дві частини після обривання сторінки.
  • orphans: Визначає мінімальну кількість рядків тексту, які мають залишатися разом у верхній частині сторінки.
  • widows: Визначає мінімальну кількість рядків тексту, які мають залишатися разом у нижній частині сторінки.
  • perspective: Визначає відстань, на яку глядач спостерігає за 3D-простором.
  • perspective-origin: Визначає точку, з якої ви переглядаєте 3D-елемент.
  • object-fit: Визначає, як розміщується зображення всередині свого контейнера.
  • object-position: Визначає позицію зображення всередині свого контейнера.
  • backdrop-filter: Застосовує ефекти фільтра до зображення, яке знаходиться позаду елементу.
  • filter: Застосовує ефекти фільтра до елемента, такі як розмитесть, насиченість тощо.
  • scroll-behavior: Визначає поведінку прокрутки сторінки, коли вона переміщається до якорних ссилок.
  • scroll-snap-type: Визначає тип прив'язки елементів під час прокрутки на смартфонах і планшетах.
  • scroll-snap-align: Визначає, як точно елемент повинен вирівнюватися під час прокрутки.
  • scroll-snap-stop: Визначає, чи має прокрутка зупинятися, якщо користувач натискає кнопку.
  • clip-path: Вирізає область елемента та робить його прозорим за межами цієї області.
  • mask: Застосовує маску до елемента, визначаючи, яка частина елемента є видимою.
  • shape-outside: Визначає область, яка визначає, як текст обтікає вкладений об'єкт.
  • shape-margin: Визначає внутрішній відступ від області форми, визначеної властивістю shape-outside.
