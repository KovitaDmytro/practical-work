Адаптивний дизайн

Опис завдання

Мета цього завдання - створення веб-сторінки з адаптивним дизайном, що означає, що вміст сторінки має оптимально відображатися на різних пристроях та розмірах екранів, забезпечуючи зручний та ефективний користувацький досвід.

Виконання

Метатег viewport: Використаний метатег <meta name="viewport" content="width=device-width, initial-scale=1.0">, щоб забезпечити, що ширина вмісту відповідає ширині екрану пристрою і масштаб сторінки встановлюється на 1.0.

Медіа-запити: Використані CSS медіа-запити для адаптації вигляду сторінки до різних розмірів екрану, що дозволяє оптимізувати розміщення та відображення контенту на маленьких екранах, таких як мобільні пристрої.

Результат

Створено веб-сторінку, яка забезпечує зручний та ефективний користувацький досвід на різних пристроях та розмірах екранів.

Використання додаткових тегів для форматування тексту

Опис завдання

У веб-сторінці використовуються різні додаткові HTML теги для форматування тексту з метою забезпечення кращого його представлення та розуміння.

Використання тегів

Теги <q> та <blockquote> використані для відображення цитат та блокових цитат в контексті тексту.

Тег <abbr> використаний для відображення абревіатури з поясненням, яке відображається при наведенні курсора.

Тег <address> використаний для відображення контактних даних або адреси.

Тег <cite> використаний для відображення цитат або посилань на назву твору.

Тег <bdo> використаний для зміни напрямку тексту (зліва направо або справа наліво).

Результат

Використання цих тегів дозволяє краще структурувати та форматувати текст, зробити його більш зрозумілим і зручним для користувача.

Приклад веб-сторінки з форматуванням тексту

Ось декілька прикладів використання тегів форматування тексту:

Це приклад цитати з використанням тегу <q>.

Це блок цитати з використанням тегу <blockquote>.

ПЗ - це абревіатура.

Адреса: Київ, вул. Хрещатик, 1

Цитата з використанням тегу <cite>.

Цей текст направлений зправа наліво, використовуючи тег <odb>.
