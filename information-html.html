Вступ

HTML (HyperText Markup Language) є основною мовою для створення веб-сторінок. Вона визначає структуру та семантику вмісту веб-документа, що дозволяє браузерам правильно відображати інформацію для користувача. HTML є однією з основних складових веб-технологій і є фундаментальним елементом для розробки веб-сторінок та веб-додатків.

Історія HTML

HTML була розроблена Тімоті Бернерсом-Лі у 1991 році в рамках проекту World Wide Web (WWW) у Європейському організації ядерних досліджень (CERN). Початково HTML була простою мовою для створення гіпертекстових документів, яка складалася з набору тегів, що визначали структуру тексту та посилання між сторінками.

Розвиток HTML

Протягом років HTML зазнавала численних змін і вдосконалень. Послідовні версії мови додавали нові функції, елементи та можливості, що розширювали її функціональність. Однією з ключових подій у розвитку HTML було введення стандарту HTML5 у 2014 році, який приніс значні зміни і нові можливості для веб-розробників.

Роль HTML у веб-розробці

HTML відіграє важливу роль у веб-розробці, визначаючи структуру та семантику веб-сторінок. Вона дозволяє розробникам створювати доступні та ефективні веб-додатки, які можуть взаємодіяти з користувачами через різноманітні пристрої та платформи.

Приклади в кожному розділі


    <!DOCTYPE html>
    <html>
    <head>
    <title>Назва сторінки</title>
    </head>
    <body>

    <h2>Мій перший заголовок</h1>
    <p>Мій перший параграф.</p>

    </body>
    </html>
